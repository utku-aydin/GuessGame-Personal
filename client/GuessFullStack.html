This is a number guessing game known as "Bulls and Cows".
In each game, a 4-digit number is generated where every digit is different.
For each round, you guess a number and are told the exact and partial digit matches.

An exact match occurs when you guess the correct digit in the correct position.
A partial match occurs when the you guess the correct digit but in the wrong position.
Once the number is guessed (exact matches for all digits) you win the game.
